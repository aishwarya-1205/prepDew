# PrepDew - AI-Powered Interview Preparation Platform

A comprehensive full-stack web application that revolutionizes interview preparation by providing personalized, AI-generated questions and practice sessions tailored to specific roles and technologies.

## 🚀 Key Features

- **Personalized Session Creation**: Generate custom interview sessions based on role, experience level, and technology stack
- **AI-Powered Question Generation**: Leverage advanced AI to create relevant, industry-standard interview questions
- **Interactive Practice Environment**: Simulate real interview scenarios with timed practice sessions
- **Progress Tracking**: Monitor preparation progress and performance analytics
- **Role-Specific Content**: Tailored questions for Full Stack Developer, Frontend, Backend, and other tech roles
- **Technology Focus**: Specialized questions for React, Node.js, MongoDB, Express, JavaScript ES6+, and more

## 🛠 Tech Stack

**Frontend:**
- React.js with Hooks
- Tailwind CSS for responsive design
- React Router for navigation
- Axios for API integration
- React Hot Toast for notifications
- Moment.js for date handling
- React Markdown for content rendering
- Syntax highlighting with Prism

**Backend:**
- Node.js & Express.js
- MongoDB with Mongoose ODM
- JWT Authentication
- RESTful API architecture
- Session management

## 💡 What Makes It Special

- **Smart Question Generation**: AI algorithms analyze job requirements and generate contextually relevant questions
- **Adaptive Difficulty**: Questions adjust based on experience level (Junior, Mid, Senior)
- **Comprehensive Coverage**: Covers technical concepts, coding challenges, system design, and behavioral questions
- **Clean UI/UX**: Modern, intuitive interface designed for optimal learning experience
- **Scalable Architecture**: Built with modular components and clean code practices
